# recon-python

Tool for recon domain/ip

by using these tool:
    ping
    nmap

Running script as root, will use -sV (determine open ports service version) and -osscan-guess (guess the os of target)

Usage:
    
    chmod +x ./recon.py
    
------- 
    
    ./recon.py <target> 
-
    : default recon run

------- 

    sudo ./recon.py <target>
-
    : advanced recon run
